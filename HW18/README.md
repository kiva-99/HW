# Домашнее задание №18 (HW-18)

## Цель
Работа с реляционной базой данных MySQL:
- Создание таблиц и заполнение данными
- Выполнение SQL-запросов с использованием JOIN
- Резервное копирование и восстановление базы данных
- Автоматизация бэкапов с помощью cron

## Выполненные задачи

### Задание 1
Выведен список анализов, проданных с 5 по 12 февраля 2020 года включительно:
- Общий анализ крови
- Глюкоза крови
- Антитела к SARS-CoV-2
- Билирубин общий
- Лейкоцитарная формула

### Задание 2 (опциональное)
Созданы таблицы `Students` и `Courses`, выполнен запрос с `LEFT JOIN` для отображения всех студентов и их курсов (включая `NULL` при отсутствии курса).

### Задание 3
- Создан SQL-бэкап базы данных `hw18`
- Выполнено тестовое изменение данных и последующее восстановление из бэкапа
- Написан скрипт `backup_hw18.sh` для автоматического создания бэкапов
- Настроен `cron` на ежедневный запуск скрипта в 02:00
