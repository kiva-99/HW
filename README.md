# HW 

## hw12
- `1.c` — C-программа, выводящая "HELLO Ubuntu".
- `args.sh` — выводит все аргументы командной строки на консоль и в файл.
- `find_files.sh` — ищет файлы с заданным расширением в указанном каталоге и сохраняет пути в файл.
- `file_info.sh` *(опциональное)* — выводит права доступа и размер всех файлов в каталоге и подкаталогах.
- `search_string.sh` *(опциональное)* — ищет заданную строку в файлах каталога, выводит путь и размер найденных файлов.

## hw13
- `count_words.sh` — считает количество слов в тексте. 
- `fix_repeats.sh` — удаляет повторяющиеся слова, разделённые пробелами.
