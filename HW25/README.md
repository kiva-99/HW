\## Задание 1: Создание образа через Dockerfile



\### Требования из задания

\- Базовый образ: `ubuntu:20.04`

\- Установка пакетов: `nginx`, `curl`

\- Каталог для статики: `/app/static`

\- Порт: `8080`

\- Отдача статических файлов



\### Инструкции для воспроизведения




\# Перейти в директорию задания 1

cd task1



\# Сборка образа

docker build -t hw25-webserver:latest .



\# Запуск контейнера

docker run -d -p 8080:8080 --name hw25-container hw25-webserver:latest



\# Проверка работы

curl http://localhost:8080

\# Или открыть в браузере: http://localhost:8080
