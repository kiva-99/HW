## 📚 Задания
# Задание 1: Класс "Круг"
Атрибуты: радиус, цвет
Методы: вычисление площади, вычисление длины окружности
Принципы: Класс, объект, __init__, self
# Задание 2: Класс "Банковский счет"
Атрибуты: номер счета, имя владельца, баланс
Методы: пополнение (deposit), снятие (withdraw)
Принципы: Инкапсуляция, валидация данных
# Задание 3: Класс "Студент"
Атрибуты: имя, возраст, средний балл
Методы: вычисление GPA, определение статуса
Принципы: Списки, логика if/elif/else
# Задание 4: Класс "Книга"
Атрибуты: название, автор, год издания
Методы: Getters, Setters
Принципы: Приватные атрибуты (__), защита данных
# Задание 5: Класс "Автомобиль"
Атрибуты: марка, модель, цвет, год выпуска
Методы: Getters, Setters
Принципы: Полная инкапсуляция, валидация
