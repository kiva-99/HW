# HW27 — Работа с основными структурами данных в Python

## Описание
Домашнее задание по темам: списки, кортежи, множества, строки, файлы, базовые алгоритмы.

## Структура проекта
```
HW27/
├── task1.py # Проверка доступности IP-адресов через ping
├── task2.py # Подсчёт символов в строке (регистр, цифры, пунктуация)
├── task3.py # Поиск общих элементов двух списков
├── task4.py # Сортировка чисел по убыванию
├── task5.py # Проверка уникальности элементов кортежа
├── task6.py # Поиск файлов по подстроке в имени
├── task7.py # Поиск общих символов двух строк
├── task8.py # Вычисление медианы списка чисел
├── task9.py # Замена гласных букв на символ "-"
├── task10.py # Уникальные общие элементы двух списков
├── ping_results.txt # Пример результата task1.py
└── README.md # Этот файл
```

## Как запустить

python taskN.py  # где N — номер задания (1-10)
