# Домашнее задание №28 — Применение функций, работы с файлами/директориями и шаблонизатора Jinja2

##  Описание
Конечный результат это гармоничное приложение, объединяющее все изученные темы:
- Функции (умножение чисел для защиты от ботов)
- Работа с файлами (логирование действий)
- Работа с директориями (создание папки `logs/`)
- Шаблонизатор Jinja2 (генерация HTML-страницы со списком пользователей)

##  Функционал
1. Пользователь проходит проверку "не робот":
   - Вводит два числа от 0 до 9
   - Решает пример умножения с помощью функции `multiply_numbers()`
2. Все действия записываются в лог-файлы в папке `logs/`
3. После успешной проверки генерируется красивая HTML-страница со списком пользователей
4. Страница автоматически открывается в браузере
