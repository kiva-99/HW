\# HW-26: Основы Python



\## Выполненные пункты

### Теоретическая часть
1. **История Python** — изучено: создатель Гвидо ван Россум (1989–1991 гг.), название от шоу «Monty Python's Flying Circus», версии 2.x (устарела) и 3.x (актуальная).
2. **Установка Python** — установлена версия 3.14.3, проверена командой `python --version`.
3. **Типы данных** — изучены и проверены: `int` (целые), `str` (строки), `float` (дробные) через функцию `type()`.

### Практическая часть
4. `hello.py` — выводит фиксированное сообщение «Привет, мир!» через `print()`.
5. `calc.py` — вычисляет сумму `2 + 2 = 4` с выводом результата.
6. `greet_name.py` — запрашивает имя пользователя через `input()` и выводит персонализированное приветствие.
7. `count_1_to_10.py` — выводит числа от 1 до 10 с помощью цикла `for` и функции `range(1, 11)`.
8. `age_info.py` — запрашивает возраст и выводит сообщение с использованием f-строки.
9. `multiply_two.py` — запрашивает два числа, вычисляет произведение, выводит с форматированием до 2 знаков (`{:.2f}`).
10. `first_letter.py` — запрашивает слово и выводит первую букву через индексацию строки `[0]` с защитой от пустого ввода.
11. `square_num.py` — запрашивает целое число и выводит его квадрат через оператор возведения в степень `**`.
12. `multiplication_table_5.py` — выводит таблицу умножения на 5 (от 1 до 10) с использованием цикла `for`.
13. `average_two.py` — запрашивает два числа и вычисляет их среднее арифметическое `(a + b) / 2` с форматированием вывода.
